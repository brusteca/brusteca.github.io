Videogame projects

LIGHTBALL

2021

An Arkanoid clone with some space shooter thrown in for good measure. Featuring 3D graphics, procedurally generated levels, a lot of lights and wacky sounds. It's playable in the browser, but it runs best as a download.

The Tale of Zulma

2021

A short demo of a Zelda-like game I made in order to learn Unity. I wanted to get a hold of Unity basics with something that had simple mechanics and no physics. Playable in the browser!

ESTAR WORDS

2017

A scripted demo of a space battle created using WebGL. This project was created for the Advanced Computer Graphics class at FIng, UdelaR alongside a partner. We built our own engine in JavaScript and implemented model loading, mapped terrain, simple physics, particles, dynamic lights and even some shadows.
